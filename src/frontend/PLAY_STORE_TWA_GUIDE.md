# Publishing Snake Game to Google Play Store

This guide explains how to package your Snake Game PWA as an Android App Bundle (AAB) using Trusted Web Activity (TWA) and Bubblewrap, then submit it to the Google Play Store.

## Prerequisites

Before you begin, ensure you have:

1. **Node.js and npm** installed (version 14 or higher)
2. **Java Development Kit (JDK)** installed (version 8 or higher)
3. **Android SDK** with command-line tools
4. **Google Play Developer Account** ($25 one-time registration fee)
5. Your Snake Game **deployed and accessible via HTTPS** (already done via Caffeine deployment)

## Step 1: Install Bubblewrap

Bubblewrap is Google's official tool for generating Android apps from PWAs.
